# Nexus Finance Frontend

## Overview
The frontend is a React-based application built with Vite and Tailwind CSS. It provides a futuristic, agentic commerce OS interface for managing DeFi and Real-World Assets (RWA).

## Architecture
- **`src/pages/`**: Main application views (Landing, Dashboard, AgentOS, etc.).
- **`src/components/`**: Reusable UI components (Sidebar, Topbar, Modals).
- **`src/lib/`**: External service configurations (Supabase, AI).
- **`src/styles/`**: Global CSS and theme configurations.
- **`src/types/`**: TypeScript interfaces and enums.

## Setup
1. Ensure environment variables are configured in `.env`.
2. Run `npm install` from the root.
3. Run `npm run dev` to start the development server.

## Design Philosophy
- **Futuristic Aesthetic**: Dark theme with neon accents, grid patterns, and scanline effects.
- **Agent-First**: Centered around AI agents (ARIA, DELTA, SIGMA, KAPPA) that automate DeFi tasks.
